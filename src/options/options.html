Job match scoring

Scores every job page against your resumes ("Software development: 80%, Project management: 30%") so you know which resume to attach. A free offline keyword estimate always works; connect your own AI key for much smarter semantic scores. Privacy note: with AI enabled, the job description and your resume text are sent to the provider you configure below. Your API key is stored encrypted in the vault.

Cheap/fast models are enough for scoring — e.g. google/gemini-2.0-flash-lite-001 or deepseek/deepseek-chat on OpenRouter, gpt-4o-mini on OpenAI, claude-haiku-4-5-20251001 on Anthropic, gemini-3.1-flash-lite on Google. For Google (Gemini), only the API key is needed — the endpoint and a default model are set automatically per the official docs. Results are cached per job, so re-opening a job costs nothing.
